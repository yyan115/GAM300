[DEFINITIONS]
- Panel contains a set of board cutscene images
- Board refers to a cutscene image


[CONFIGS]
- Display Board Duration: 3
  - DESC: How long an image stays on screen before switching to the next image in the same panel

- Display Panel Duration: 6
  - DESC: How long before fading to the next panel if it is the last image in the panel

- Fade Duration: 0.5
  - DESC: How long it fades in and out to the next image

- Skip Fade Duration: 1.0
  - DESC: How long it fades to level scene after skipping OR if last panel and image

- Typewriter Text Speed: 0.02
  - DESC: How fast a text character appears before the next
